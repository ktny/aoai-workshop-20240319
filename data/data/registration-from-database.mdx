## Oracle

Oracle Databaseに接続する場合は、JDBCドライバーを使用して接続します。\
その際の手順は以下のようになります。KIの画面上から操作可能です。

1. KIの画面からJDBCドライバーをアップロード\
   \
   管理者でログインします。\
   システム設定からドライバーアップロードに移り、[ファイル選択]から「ojdbc8.jar」をアップロードしてください。\
   JDBCドライバーの取得については以下のページを参照してください。\
   [https://www.oracle.com/database/technologies/appdev/jdbc-downloads.html](https://www.oracle.com/database/technologies/appdev/jdbc-downloads.html)
   <Img src="/assets/img/capture/capture-various-databases-1.png" alt="ドライバーアップロード" client:load />
1. データベース接続情報の入力\
   \
   [データセット一覧]画面で[データベース接続]ボタンをクリックします。\
   表示された画面にて、保存済みの接続設定を選択するか、以下の項目を設定してください。

   | 項目名             | 説明                                      |
   | ------------------ | ----------------------------------------- |
   | 接続先データベース | プルダウンから「Oracle」を選択します。    |
   | ホスト名           | Oracle Databaseのホスト名を入力します。   |
   | ポート番号         | Oracle Databaseのポート番号を入力します。 |
   | サービス名         | Oracle Databaseのサービス名を入力します。 |
   | ユーザー名         | Oracle Databaseのユーザー名を入力します。 |
   | パスワード         | Oracle Databaseのパスワードを入力します。 |

1. テーブルの選択:version[2.1.5]\
   [データベースからのインポート - ②テーブル選択]画面が表示されますので、以下の項目を指定して[インポート]ボタンをクリックしてください。
   - [データセット名]：\
      インポート後にKI上で表示する名前を入力します。
   - [テーブル選択方法]：\
     データセットとしてインポートするテーブルの選択方法を、[プルダウンで選択]、または[SQL入力]のいずれかから選択します。\
      選択方法に応じて、必要な情報を指定してください。
     - [プルダウンで選択]を選択した場合：\
        [テーブル]のプルダウンからインポートするテーブルを選択してください。\
        画面下部にテーブルのプレビューが表示されます。
     - [SQL入力]を選択した場合：\
        [SQL]にSQL文を入力してください。\
        [プレビュー]ボタンをクリックすると、画面下部に取得対象データのプレビューが表示されます。

## Salesforce

Salesforceへの接続方法について説明します。

1. データベース接続情報の入力\
    \
    [データセット一覧]画面で[データベース接続]ボタンをクリックします。\
    表示された画面にて、保存済みの接続設定を選択するか、以下の項目を設定してください。

   | 項目名               | 説明                                           |
   | -------------------- | ---------------------------------------------- |
   | 接続先データベース   | プルダウンから「Salesforce」を選択します。     |
   | ホスト名             | Salesforceのホスト名を入力します。             |
   | ユーザー名           | Salesforceのユーザー名を入力します。           |
   | パスワード           | Salesforceのパスワードを入力します。           |
   | セキュリティトークン | Salesforceのセキュリティトークンを入力します。 |

1. テーブルの選択:version[2.1.5]:version[2.1.6]\
   \
   [データベースからのインポート - ②テーブル選択]画面が表示されますので、以下の項目を指定して[インポート]ボタンをクリックしてください。
   - [テーブル]：\
      インポートするテーブルをプルダウンから選択します。\
      画面下部にテーブルのプレビューが表示されますので、インポートする列をチェックボックスから選択してください。
   - [データセット名]：\
      インポート後にKI上で表示する名前を入力します。
   - [高度な設定]：\
      インポートする行を絞り込みたい場合に設定します。\
      [SOQL で条件入力]にSOQL文で条件を入力してください。\
      [プレビュー]ボタンをクリックすると、条件に合致するデータを画面下部から確認できます。

   :::note{.point}
   ポイント

   インポートにSalesforce Bulk APIを利用しており、テーブルの選択やSOQLによる条件指定において以下の制限があります。

   - \*EventRelationなど、一部のテーブル(sObject)はサポートしておりません。\
     詳細については、以下のページを参照してください。\
     [https://help.salesforce.com/s/articleView?id=000383508&language=ja&type=1](https://help.salesforce.com/s/articleView?id=000383508&language=ja&type=1)
   - OFFSETなど、一部のSOQLはサポートしておりません。\
      詳細については、以下のページを参照してください。\
     [https://developer.salesforce.com/docs/atlas.ja-jp.212.0.api_asynch.meta/api_asynch/asynch_api_concepts_limits.htm](https://developer.salesforce.com/docs/atlas.ja-jp.212.0.api_asynch.meta/api_asynch/asynch_api_concepts_limits.htm)
   - FOR句はサポートしておりません。
   :::

   [データベースからのインポート - ②テーブル選択]画面が表示されますので、以下の項目を指定して[インポート]ボタンをクリックしてください。
   - [データセット名]：\
      インポート後にKI上で表示する名前を入力します。
   - [テーブル選択方法]：\
      データセットとしてインポートするテーブルの選択方法を、[プルダウンで選択]、または[SQL入力]のいずれかから選択します。\
      選択方法に応じて、必要な情報を指定してください。
     - [プルダウンで選択]を選択した場合：\
        [テーブル]のプルダウンからインポートするテーブルを選択してください。\
        画面下部にテーブルのプレビューが表示されます。
     - [SQL入力]を選択した場合：\
        [SQL]にSQL文を入力してください。\
        [プレビュー]ボタンをクリックすると、画面下部に取得対象データのプレビューが表示されます。

   :::note{.hint}
   ヒント

   外部データベースとしてBigQueryを追加した際に、BigQuery APIレスポンスが10GBを超えるなどBigQuery APIの制限([https://cloud.google.com/bigquery/quotas?hl=ja](https://cloud.google.com/bigquery/quotas?hl=ja))を超過した場合はエラーが表示されることがあります。\
   その際は、[フィルター設定]にて[SQL入力]を選択し、取得する列の制限やLIMIT句を用いた行数の制限をおこなうことをご検討ください。
   :::

1. ファイルの選択 \
   \
   [データベースからのインポート - ②ファイル選択]画面が表示されますので、以下の項目を指定して[インポート]ボタンをクリックしてください。

   - [プレフィックス]：\
      取得するファイルのプレフィックスを入力します。\
      プレフィックスを指定することで、[ファイル]のプルダウンに表示される一覧を絞り込むことができます。
   - [ファイル]：:version[2.4.0]\
      プルダウンからインポートするファイルを選択します。\
      ファイルは最大1000件まで表示されます。\
      ファイルを選択すると画面下部にプレビューが表示されます。
      - インポート時のデータ型をヘッダーのプルダウンから選択できます。\
        指定したデータ型に変換できないデータが含まれている場合は、文字列型に変更されます。\
        強制的にデータ型を変更する場合は、[例外を無視してデータ型を変更する]にチェックを入れてください。例外データはNULL(空欄)に変換されます。
      - インポートする列をヘッダーのチェックボックスから選択してください。\
        [すべてチェック]ボタンをクリックすると、すべての列にチェックを入れることができます。\
        [すべてチェックを外す]ボタンをクリックすると、すべての列からチェックをはずすことができます。
   - [データセット名]：\
      インポート後にKI上で表示する名前を入力します。
   - [高度な設定]：\
      ヘッダーに関する設定をおこないます。

     - Excelファイル(xls、 xlsx)：

       | 項目           | 説明                                               |
       | -------------- | -------------------------------------------------- |
       | ヘッダー行番号 | ヘッダー(列名)となる行番号を指定します。           |
       | ワークシート   | インポートするシート名をプルダウンから選択します。 |

     - CSVファイル/TSVファイル/TXTファイル：

       | 項目           | 説明                                                   |
       | -------------- | ------------------------------------------------------ |
       | ヘッダー行番号 | ヘッダー(列名)となる行番号を指定します。               |
       | 引用符         | データで使用している引用符をプルダウンから選択します。 |
       | 区切り文字     | データの区切り文字をプルダウンから選択します。         |
       | エンコード     | エンコードをプルダウンから選択します。                 |
