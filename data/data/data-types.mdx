---
title: データ型の種類
---

KIで使用できるデータ型の種類は、以下のとおりです。
| データ型 | アイコン                                                   | 説明                                                         |
| -------- | ---------------------------------------------------------- | -------------------------------------------------------------|
| 文字列型 | :icon[/assets/img/common/data-string.svg,文字列型]         | 文字列のデータ型です。                                       |
| 整数型   | :icon[/assets/img/common/data-integer.svg,整数型]          | 整数のデータ型です。                                         |
|          |                                                            | 範囲は-9223372036854775808～9223372036854775807              |
| 小数型   | :icon[/assets/img/common/data-decimal.svg,小数型]          | 小数点を含む数値のデータ型です。                             |
|          |                                                            | 範囲は15桁精度です。                                         |
| 日時型   | :big-icon[/assets/img/common/data-datetime.svg,日時型]     | 日付と時刻のデータ型です。                                   |
| 日付型   | :big-icon[/assets/img/common/data-date.svg,日付型]         | 日付(時刻なし)のデータ型です。                               |
| 時刻型   | :big-icon[/assets/img/common/data-time.svg,時間型]         | 時刻(日付なし)のデータ型です。                               |
| 論理型   | :icon[/assets/img/common/data-boolean.svg,論理型]          | True(真)、またはfalse(偽)のいずれかの値をとるデータ型です。  |
